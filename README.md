# ymos-diagnosis — 投资策略诊断

**不推荐股票，不预测走势，只诊断你的投资"操作系统"。**

## ymos-diagnosis 解决什么问题

独立投资者最常见的困境：**知道很多道理，但一到实操就变形。**

- 明知道该止损，但"再等等"
- 说好了不追高，但怕错过
- 买之前很冷静，买之后很焦虑
- 赚了觉得自己牛，亏了觉得市场错

这些问题的根源不是知识不够，是投资系统有漏洞——而你可能根本不知道漏洞在哪里。

ymos-diagnosis 帮你做三件事：

| 能力 | 描述 |
|:---|:---|
| **梳理投资思路** | 你的策略到底是什么？说得出来吗？能复现吗？ |
| **审视投资逻辑** | 你的每个判断背后有没有隐含的错误假设？有没有把运气当实力？ |
| **体检投资系统** | 策略、风控、仓位、情绪、Beta 认知——7 项检验逐项过一遍，出诊断报告 |

## 两种工作模式

### 问诊模式

你带着一个具体的投资问题来（"该不该买 XX""我被套了怎么办""这个板块还能追吗"），AI 不会直接回答，而是先帮你判断**这个问题本身成不成立**。

工作方式是一个 5 层消解漏斗，逐层过滤：

| 层 | 检测什么 | 举例 |
|:---|:---------|:-----|
| 第 1 层 | 语言陷阱 | "这是好股票"——"好"的定义是什么？ |
| 第 2 层 | 假设错误 | "跌了这么多应该反弹"——下跌不是反弹的理由 |
| 第 3 层 | 逻辑错误 | "我上次买赚了所以这次也行"——上次赚是运气还是逻辑？ |
| 第 4 层 | 事实前提 | "15 倍 PE 很便宜"——便宜是相对什么？行业均值？历史？ |
| 第 5 层 | 信息充分 | "该不该买 XX"——止损线定了吗？仓位多少？盈亏比算了吗？ |

大部分投资问题会在前 3 层被消解掉——因为问题本身就是错的。剩下的才是真问题，AI 会帮你正面解答。

### 体检模式（7 项投资检验）

你没有具体问题，但想让 AI 用一套框架把你的投资系统拆一遍。7 项检验逐项进行，每做完一项都会停下来跟你讨论，最后出一份完整的诊断报告。

| # | 检验 | 问什么 |
|:---|:------|:---------|
| 1 | 策略存在性 | 你有没有一个能写成一段话的投资策略？写不出来 = 没有策略 |
| 2 | 可复现性 | 过去赚钱的交易，逻辑能复现吗？还是靠的那个时间点的运气？ |
| 3 | 风控存在性 | 你有止损规则吗？上次止损是什么时候？ |
| 4 | 仓位纪律 | 你最大的单只持仓占比多少？你定过上限吗？ |
| 5 | 情绪审计 | 你最近一次冲动交易是什么时候？触发条件是什么？ |
| 6 | Beta 认知 | 你知道你持仓的 Beta 在多少吗？你赚的是 Alpha 还是随大盘涨？ |
| 7 | 成长层级 | 你现在是"看别人买什么跟着买"、还是"有自己的逻辑但不稳定"、还是"有系统在跑"？ |

诊断报告会给出每项的健康度评估、发现的问题、以及具体的改进建议。

## 核心理念：6 条投资公理

这 6 条公理是整个诊断框架的基石。AI 的每一个判断都可以追溯到这些公理：

1. **策略必须可复述、可复制** — 说不出来的不叫策略，叫运气。可复现的逻辑才是资产，不可复现的结果只是噪声
2. **PVP 和 PVE 是完全不同的游戏** — PVE（增量市场）可以长持，PVP（存量博弈）只能短打。用错规则必输
3. **先板块 Beta，再个股 Alpha** — 板块趋势决定 80% 的收益。顺序反了等于赌博
4. **情绪是最大的交易成本** — 系统的意义不是帮你赚更多，是帮你在情绪出现时用流程替代判断
5. **赔率 x 概率 = 真实期望值** — 只看赔率不看概率是赌徒思维
6. **99% 的投资亏损是心理问题** — 不是不知道该止损，是"舍不得"

## 安装指南

### 方式一：Claude Code（推荐）

```bash
git clone https://github.com/Evan-XYZ/ymos-diagnosis.git /tmp/ymos-diagnosis \
  && cp -r /tmp/ymos-diagnosis/skills/ymos-diagnosis ~/.claude/skills/ \
  && rm -rf /tmp/ymos-diagnosis
```

安装完成后，在 Claude Code 中输入以下任一触发词即可启动：

```
/ymos-diagnosis
/投资诊断
帮我看看我的投资
诊断一下我的策略
```

AI 会让你选择「问诊」还是「体检」，然后一步步带你走完。

### 方式二：其他 AI Agent 工具

ymos-diagnosis 可以在任何支持读取文件的 AI Agent 环境中运行——Claude Code、ChatGPT Codex、OpenClaw（龙虾）等都可以。

**通用方法**：

1. 下载项目到本地：
   ```bash
   git clone https://github.com/Evan-XYZ/ymos-diagnosis.git
   ```

2. 告诉你的 AI Agent 读取 SKILL.md：
   ```
   请读取这个文件：/你的本地路径/ymos-diagnosis/skills/ymos-diagnosis/SKILL.md

   然后按照里面的指令为我做投资诊断
   ```

3. AI 会自动进入诊断模式，让你选择问诊或体检

> SKILL.md 是完全自包含的——它既是 AI 的操作手册，也是完整的诊断框架。只要 AI 能读到这个文件，就能运行诊断。

### 方式三：直接使用知识库文件

如果你不用任何 Agent 工具，knowledge/ 目录下的文件也可以独立使用：

- 把 `investment_axioms_and_framework.md` 粘贴到任何 AI 的 system prompt → 你的 AI 就有了投资纪律审计能力
- 直接阅读 `diagnosis_case_library.md` → 看看你有没有踩过里面的坑

## 使用指南

### 第一步：启动诊断

启动后 AI 会问你选哪种模式：

> **问诊** — 带着具体问题来，帮你判断问题本身成不成立
> **体检** — 没有具体问题，用框架把你的投资系统拆一遍

### 第二步：跟 AI 对话

**问诊模式**：直接说出你的投资问题。AI 会逐层消解（语言陷阱 → 假设错误 → 逻辑错误 → 事实前提 → 信息充分），每消解一层都会停下来跟你讨论。不是一次性给答案，而是帮你自己想清楚。

**体检模式**：AI 会先收集你的基本信息（投资经验、主要持仓、资金规模、投资目标、风格偏好、近期操作），然后逐项进行 7 项检验。每做完一项会停下来告诉你结论，等你回应后再继续下一项。

### 第三步：获得结论

**问诊模式**：大部分问题会在消解过程中被"解决"——因为你会发现问题本身是错的，或者问题的答案在消解过程中已经浮现。真正需要回答的问题，AI 会基于公理框架正面解答。

**体检模式**：7 项检验全部完成后，AI 会出一份诊断报告，包含每项的健康度、发现的问题、以及改进建议。

### 使用建议

- **诚实回答**：诊断的质量取决于你提供的信息真实度。AI 不会评判你，但如果你隐瞒信息，诊断就不准
- **不要急**：每一步的对话都值得认真想。投资诊断不是填问卷，是一次深度的策略审视
- **定期体检**：建议每季度做一次体检，特别是在大幅盈利或亏损之后

| 文件 | 角色 | 说明 |
|:---|:---|:---|
| `SKILL.md` | 总入口 | AI 读这一个文件就能运行完整诊断。包含角色定义、6 条公理、模式选择、问诊漏斗、体检检验、说话风格、案例库 |

## 常见问题

**Q：需要会写代码吗？**

不需要。安装完成后，跟 AI 说话就行。所有操作都是对话式的。

**Q：AI 会推荐股票吗？**

不会。ymos-diagnosis 严格只做诊断——诊断你的策略、逻辑、纪律、情绪管理。它不会告诉你该买什么、该卖什么、什么时候操作。

**Q：支持哪些 AI 工具？**

任何能读取文件的 AI Agent 都可以。推荐方案：
- **Claude Code** — 原生支持 Skill 系统，安装后输入 `/ymos-diagnosis` 即可
- **ChatGPT Codex** — 让它读取 SKILL.md 文件
- **OpenClaw（龙虾）** — 本地运行，数据不出本机
- **其他 Agent** — 只要能读文件，把 SKILL.md 喂给它就行

**Q：诊断结果准吗？**

诊断质量取决于两个因素：你提供的信息是否真实完整，以及底层 AI 模型的能力。建议使用当下最强的模型（如 Claude Opus 4.6 或 ChatGPT-5.4）来获得最好的诊断效果。

**Q：knowledge/ 文件夹是必须的吗？**

不是必须的。SKILL.md 是完全自包含的，没有 knowledge/ 文件也能运行诊断。但如果你把 knowledge/ 目录也给 AI 读取，诊断会更深入——AI 能引用更详细的框架解释和真实案例。

**Q：我的投资信息安全吗？**

ymos-diagnosis 是纯本地运行的 Markdown 文件，不收集任何数据、不上传任何信息。你的对话内容受限于你使用的 AI 工具本身的隐私政策。

**Q：体检完发现问题很多怎么办？**

这是正常的。大部分投资者的系统都有漏洞，区别只在于知不知道。知道了才能改。诊断报告会给出具体的改进建议，你可以按优先级逐项修复。如果你想把梳理出来的策略系统化，可以看看 [YMOS](https://github.com/Evan-XYZ/YMOS)。
